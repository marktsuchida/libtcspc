# libtcspc Python documentation

```{warning}
The libtcspc Python bindings are currently under development and are not
ready for general use.
```

The `libtcspc` Python package provides live processing for time-correlated
single-photon counting (TCSPC) and time-tags streams. Processing graphs are
described in Python, generated as C++ source, and compiled at runtime into
optimized machine code.
